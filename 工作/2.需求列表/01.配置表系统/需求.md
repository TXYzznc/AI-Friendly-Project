# 需求 - 01.配置表系统

> 来源：[策划案.md](../../1.策划/01.配置表系统/策划案.md)

## 开发需求清单

### 1. 框架机制代码（Assets/Scripts/Modules/DataTable/）
- `IDataTable` 接口：定义 `Load(string json)`，每个生成的 Table 容器类实现此接口
- `DataTableFile<TRow>`：表文件的运行时反序列化结构（仅关心 `rows` 字段）
- `DataTableModule`：`IGameModule` 实现
  - `ModuleCategory = 0`，`Dependencies = Type.EmptyTypes`
  - `InitializeAsync`：遍历 `DataTableRegistry.Entries`，`Resources.LoadAsync<TextAsset>` 加载每张表的 JSON，`Activator.CreateInstance` 创建 Table 容器并调用 `Load`
  - `GetTable<T>() where T : IDataTable`：返回已加载的表实例，未找到抛 `KeyNotFoundException`
  - 关键节点按 `FrameworkLogger` 规范记录日志（开始/完成/失败）

### 2. Editor 生成器（Assets/Scripts/Modules/DataTable/Editor/）
- `DataTableGenerator`：菜单项触发（如 `Tools/DataTable/生成全部配置表代码`）
- 扫描 `Assets/Resources/DataTable/*.json`
- 对每张表生成 `Assets/Scripts/DataTable/{TableName}.cs`：
  - `{TableName}Row` 类：按 `fields` 生成公开自动属性
  - `{TableName}` 类：实现 `IDataTable`，内部 `Dictionary<int, {TableName}Row>`，提供 `GetById`/`TryGetById`/`All`
- 生成/覆盖 `Assets/Scripts/DataTable/DataTableRegistry.cs`：汇总 `(文件名, TableType)` 列表
- 校验：
  - 字段类型必须在支持列表内，否则报错并中止
  - 字段名必须是合法 C# 标识符
  - `fields[0].name` 必须是 `Id` 且类型为 `int`

### 3. 示例配置表
- `Assets/Resources/DataTable/ItemTable.json`：字段 `Id(int) / Name(string) / IconId(int) / Tags(int[])`，3~4 行示例数据

### 4. GameApp 接入
- 在 `GameApp.cs` 中注册 `DataTableModule`（替换原 TODO 注释之一）
- 验证：启动后 `GetModule<DataTableModule>().GetTable<ItemTable>().GetById(1001)` 能正确返回数据

### 5. 文档清理
- 更新/合并 `Assets/DataTable/README.md` 与 `Assets/Resources/DataTable/README.md`（当前内容是旧 xlsx/.bytes 方案，需替换为新流程说明）

## 接口设计摘要

```csharp
public interface IDataTable
{
    void Load(string json);
}

public sealed class DataTableFile<TRow>
{
    public List<TRow> rows;
}

public sealed class DataTableModule : IGameModule
{
    public int ModuleCategory => 0;
    public Type[] Dependencies => Type.EmptyTypes;
    public UniTask InitializeAsync(CancellationToken ct = default);
    public UniTask ShutdownAsync(CancellationToken ct = default);
    public T GetTable<T>() where T : IDataTable;
}

// 生成产物示例
public sealed class ItemTableRow
{
    public int Id { get; set; }
    public string Name { get; set; }
    public int IconId { get; set; }
    public int[] Tags { get; set; }
}

public sealed class ItemTable : IDataTable
{
    public void Load(string json);
    public ItemTableRow GetById(int id);
    public bool TryGetById(int id, out ItemTableRow row);
    public IReadOnlyDictionary<int, ItemTableRow> All { get; }
}
```
