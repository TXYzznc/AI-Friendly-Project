## 测试策略

**自动化测试**：✅ 是

## 自动化测试脚本

**位置**：`Assets/Tests/InventoryTests/`

**脚本列表**：
- `InventoryAddItemTests.cs` - 添加物品测试
- `InventoryRemoveItemTests.cs` - 删除物品测试
- `InventoryCapacityTests.cs` - 容量管理测试
- `InventoryUITests.cs` - UI 交互测试
- `InventoryPersistenceTests.cs` - 数据持久化测试
- `InventoryFilterTests.cs` - 筛选排序测试（可选）

**测试框架**：Unity Test Framework (UTF)

**运行命令**：
```bash
# 在 Unity Editor 中
Window → General → Test Runner → Run All
```
